# AdivinaMiNúmero - Proyecto Integrador

##  Descripción
Sistema inteligente en Python donde la computadora adivina un número pensado por el usuario (1-100) utilizando algoritmos de búsqueda eficientes. Proyecto integrador del curso de Lógica de Programación.

## Características Principales
- **Tres algoritmos de búsqueda**: Binaria, con ruido, aleatoria
- **Sistema de estadísticas**: Rendimiento y eficiencia
- **Configuración flexible**: Dificultad, rango, intentos
- **Interfaz amigable**: Menú interactivo en español
-  **Análisis de rendimiento**: Comparación con óptimo teórico
